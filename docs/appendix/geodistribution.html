Distribución Geográfica de Especies

Dispositivo dinámico de síntesis de conocimiento (FAIR/CARE / OJS) - v2.0

Autor/a

M.A.Ordano

Fecha de publicación

7 de julio de 2026

Fecha de última modificación

22 de julio de 2026

Distribución geográfica de especies: la co-existencia de especies en un espacio y tiempo brinda un marco regional para la ocurrencia de interacciones entre especies, incluyéndonos.

La co-existencia regional de especies forma un marco de co-evolución entre especies, o mosaico geográfico de co-evolución. Las especies evolucionan en un ecosistema que reúne características biológicas de las otras especies con las que interactúan regularmente (además de los “factores abióticos”).

La visualización de la distribución de especies de la vida silvestre, mediante mapas, ayuda en la interpretación multidisciplinar de una consulta.

En esta interfase puedes escribir el nombre taxonómico/sistemático (“nombre científico”) de al menos un par de especies sobre las cuales buscar tener información sobre la distribución geográfica en conjunto, y una visualización en un mapa activo en línea.

También puedes descargar los datos de ubicación geográfica correspondiente a la ficha de metadatos que puedes ver instantáneamente en la búsqueda.

⚡ Tiempo de Carga: esta versión corre 100% en tu navegador con OJS (JavaScript). La primera carga puede tardar unos segundos para un par de especies. Para más especies puede tardar unos segundos más. Si se detiene, prueba recargar la página y repite la operación. Si aparece un cartel de falta de registros, prueba reduciendo el número de datos pedidos.

Tip💡 Cómo usar este apéndice interactivo
  1. Escribe una o más especies, un nombre científico por línea (precargado con Panthera onca y Puma concolor como ejemplo).
  2. Verifica los nombres con el botón “🔎 Verificar nombres contra GBIF” — se confirma si GBIF reconoce cada nombre exactamente, si hay una coincidencia aproximada, o si no lo encuentra (en este caso, re-escribir).
  3. Define cuántos registros traer por especie con el campo “Registros máx.”
  4. Aprieta “🗺️ Sincronizar Distribución Geográfica” para consultar GBIF y ver los puntos de ocurrencia en el mapa, cada especie se identifica con un color.
  5. Explora el mapa (zoom, desplazamiento, pop-ups con el nombre al hacer clic).
  6. Descarga los datos crudos en un único archivo .csv.
Importante🔎 Si dudas del nombre científico

Para que la búsqueda funcione bien, necesitas el nombre tal como lo reconoce el backbone taxonómico de GBIF. Si tienes dudas, verifica antes en:

  • GBIF Species Search — la fuente misma que consulta esta app.
  • iNaturalist — identificación asistida por comunidad y por IA; puerta de entrada sugerida si solo tienes una foto o un nombre común.
  • Tropicos.org (Missouri Botanical Garden) — referencia nomenclatural para plantas.
  • Catalogue of Life — checklist taxonómico global de consenso.
  • Open Tree of Life — árbol filogenético navegable.

// Función para validar nombre contra GBIF
async function validarNombreGBIF(nombreEspecie) {
  const url = `https://api.gbif.org/v1/species/match?name=${encodeURIComponent(nombreEspecie)}`
  try {
    const respuesta = await fetch(url)
    const datos = await respuesta.json()
    return {
      consulta: nombreEspecie,
      encontrado: datos.matchType && datos.matchType !== "NONE",
      matchType: datos.matchType || "NONE",
      scientificName: datos.scientificName || null,
      canonicalName: datos.canonicalName || null,
      status: datos.status || null,
      error: null
    }
  } catch (error) {
    return {
      consulta: nombreEspecie,
      encontrado: false,
      matchType: "ERROR",
      error: error.message
    }
  }
}

//| output: false

// Disparador de validación reactivo
validaciones_nombres = {
  if (verificar_nombres_trigger?.timestamp) {
    return await Promise.all(
      lista_especies.map(sp => validarNombreGBIF(sp))

  }
  return []
}

// Render de alertas de validación
html_validacion_nombres = html`
  <div>
    ${validaciones_nombres.length > 0 ? validaciones_nombres.map(v => {
      if (v.matchType === "EXACT") {
        return html`<div class="bio-alert bio-alert-success">
          ✅ "${v.consulta}" → reconocido como <b>${v.scientificName}</b>
        </div>`
      } else if (v.matchType === "FUZZY" || v.matchType === "PARTIAL") {
        return html`<div class="bio-alert bio-alert-warning">
          ⚠️ "${v.consulta}" es aproximado. GBIF sugiere: <b>${v.canonicalName}</b>. Revisá antes de sincronizar.
        </div>`
      } else {
        return html`<div class="bio-alert bio-alert-danger">
          ❌ "${v.consulta}" no se encontró en GBIF. Revisá la ortografía.
        </div>`
      }
    }) : html`<div class="bio-alert bio-alert-info">Haz clic en "Verificar nombres" para validar.</div>`}
  </div>
`

// Función para obtener puntos de ocurrencia de una especie
async function obtenerOcurrenciasGBIF(nombreEspecie, limite) {
  if (!nombreEspecie || nombreEspecie.length === 0) return []
  
  const url = `https://api.gbif.org/v1/occurrence/search?scientificName=${encodeURIComponent(nombreEspecie)}&hasCoordinate=true&limit=${limite}`
  
  try {
    const respuesta = await fetch(url)
    const datos = await respuesta.json()
    
    if (!datos.results || datos.results.length === 0) return []
    
    return datos.results
      .filter(d => d.decimalLatitude && d.decimalLongitude)
      .map(d => ({
        lat: d.decimalLatitude,
        lng: d.decimalLongitude,
        scientificName: d.scientificName || nombreEspecie,
        datasetKey: d.datasetKey || "unknown",
        especie_consultada: nombreEspecie
      }))
  } catch (error) {
    console.error(`Error consultando GBIF para ${nombreEspecie}:`, error)
    return []
  }
}

//| output: false

// Disparador de sincronización reactivo
todos_los_datos = {
  if (sincronizar_trigger?.timestamp) {
    return await Promise.all(
      lista_especies.map(sp => obtenerOcurrenciasGBIF(sp, max_registros))
    ).then(resultados => resultados.flat())
  }
  return []
}

//| output: false

// Estadísticas agregadas
estadisticas = {
  const especiesUnicas = [...new Set(todos_los_datos.map(d => d.especie_consultada))]
  return {
    especies_consultadas: especiesUnicas.length,
    registros_totales: todos_los_datos.length,
    datasets_origen: [...new Set(todos_los_datos.map(d => d.datasetKey))].length
  }
}

// Datos para descarga
datos_csv_export = todos_los_datos.map(d => ({

  decimalLongitude: d.lng,
  scientificName: d.scientificName,
  datasetKey: d.datasetKey,
  especie_consultada: d.especie_consultada

// SECCIÓN 4: RENDERIZADO DEL MAPA INTERACTIVO
// ============================================================

// Contenedor del mapa
container_mapa = html`<div style="height: 450px; width: 100%; border-radius: 8px; border: 1px solid #ddd; margin: 16px 0;"></div>`

// Lógica de renderizado del mapa (reactivo, se actualiza cuando cambian los datos)
{
  // Evitar renderizar si no hay datos
  if (todos_los_datos.length === 0) {
    const mensaje = sincronizar_trigger?.timestamp
      ? "No se encontraron registros con coordenadas para estos nombres. Revisá la ortografía, probá con otro nombre científico, o puede que GBIF no tenga registros geolocalizados para esa especie."
      : 'Hacé clic en "🗺️ Sincronizar" para cargar datos en el mapa'
    container_mapa.innerHTML = `
      <div style="height: 100%; display: flex; align-items: center; justify-content: center; background: #f3f4f6; color: #6b7280; font-size: 1.1em; text-align: center; padding: 20px;">
        ${mensaje}
      </div>
    `
    invalidation.then(() => {})
    return container_mapa
  }

  // Limpiar el contenedor
  container_mapa.innerHTML = ""
  
  // Inicializar mapa
  const map = L.map(container_mapa).setView([0, 0], 2)
  
  // Capa base
  L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    attribution: '© OpenStreetMap contributors',
    maxZoom: 19
  }).addTo(map)
  
  // Paleta de colores para especies (suficiente para 20+ especies)
  const colores = [
    "#3b82f6", "#ef4444", "#10b981", "#f59e0b", "#8b5cf6", 
    "#ec4899", "#06b6d4", "#84cc16", "#f97316", "#6366f1",
    "#14b8a6", "#d946ef", "#0ea5e9", "#eab308", "#64748b",
    "#fb923c", "#4ade80", "#0d9488", "#a855f7", "#e11d48"
  ]
  
  // Mapeo de especies a colores
  const especiesUnicas = [...new Set(todos_los_datos.map(d => d.especie_consultada))]
  const mapaColores = Object.fromEntries(
    especiesUnicas.map((esp, idx) => [esp, colores[idx % colores.length]])
  )
  
  // Agregar marcadores circulares con popup
  todos_los_datos.forEach(pt => {
    if (pt.lat && pt.lng && pt.lat !== 0 && pt.lng !== 0) {
      const color = mapaColores[pt.especie_consultada]
      L.circleMarker([pt.lat, pt.lng], {
        color: color,
        fillColor: color,
        radius: 5,
        weight: 1.5,
        fillOpacity: 0.7
      })
      .addTo(map)
      .bindPopup(`
        <b style="font-size: 1.1em;">${pt.scientificName}</b><br>
        <i style="font-size: 0.9em;">Consultado como: ${pt.especie_consultada}</i><br>
        <span style="font-size: 0.8em; color: #666;">Dataset: ${pt.datasetKey}</span>
      `)
    }
  })
  
  // Leyenda de colores como control de Leaflet, superpuesta en la esquina
  // superior derecha del mapa: no oculta puntos ni tapa el zoom
  const legendControl = L.control({position: "topright"})
  legendControl.onAdd = function() {
    const div = L.DomUtil.create("div", "bio-map-legend")
    div.innerHTML = especiesUnicas.map(esp => `
      <div class="bio-map-legend-item">
        <span class="bio-map-legend-dot" style="background:${mapaColores[esp]};"></span>
        <i>${esp}</i>
      </div>
    `).join("")
    L.DomEvent.disableClickPropagation(div)
    return div
  }
  legendControl.addTo(map)
  
  // Auto-ajustar vista a los datos
  const puntosValidos = todos_los_datos.filter(pt => pt.lat && pt.lng && pt.lat !== 0 && pt.lng !== 0)
  if (puntosValidos.length > 0) {
    const bounds = L.latLngBounds(puntosValidos.map(pt => [pt.lat, pt.lng]))
    map.fitBounds(bounds, { padding: [50, 50], maxZoom: 12 })
  }
  
  // Limpiar mapa al fin del ciclo de renderizado
  invalidation.then(() => {
    map.remove()
  })
  
  return container_mapa
}

function generarCSV(datos) {
  const encabezado = ["decimalLatitude", "decimalLongitude", "scientificName", "datasetKey", "especie_consultada"]
  const filas = datos.map(d => 
    encabezado.map(col => {
      const valor = d[col] || ""
      return `"${String(valor).replace(/"/g, '""')}"`.replace(/^"(.*)"$/, (_, str) => str.includes(",") || str.includes("\n") ? `"${str}"` : str)
    }).join(",")
  )
  return [encabezado.join(","), ...filas].join("\n")
}

//| output: false

function descargarCSV() {
  console.log("botón descargar: click detectado")
  try {
    const csv = generarCSV(datos_csv_export)
    const blob = new Blob([csv], {type: "text/csv;charset=utf-8;"})
    const url = URL.createObjectURL(blob)
    const link = document.createElement("a")
    link.href = url
    link.download = `interdomestic_gbif_${new Date().toISOString().split('T')[0]}.csv`
    link.style.display = "none"
    document.body.appendChild(link)
    link.click()
    document.body.removeChild(link)
    // Revocar el blob con demora: revocarlo de inmediato es una causa
    // conocida de descargas cortadas a mitad de camino en Chromium,
    // porque .click() dispara la descarga de forma asíncrona.
    setTimeout(() => URL.revokeObjectURL(url), 1000)
    console.log("descargarCSV: OK,", datos_csv_export.length, "filas")
  } catch (error) {
    console.error("descargarCSV: FALLÓ ->", error)
    alert("No se pudo generar la descarga: " + error.message)
  }
}

//| echo: false
viewof disparador_descarga = Inputs.button("📥 Descargar datos (.csv)", {
  disabled: todos_los_datos.length === 0,
  value: null,
  reduce: () => descargarCSV()
})
