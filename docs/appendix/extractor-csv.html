Antes de descargar conjuntos de datos, es fundamental comprender de dónde vienen y bajo qué marcos operan.

El Ecosistema de la Información Ambiental

Cada país organiza su conocimiento biótico en dimensiones interconectadas. Explora las definiciones conceptuales antes de realizar tu consulta:

  1. Gobernanza Central (Gubernamental): Autoridades de aplicación que dictan políticas macro y administran áreas protegidas.
  2. Líneas Base de Conservación (Libros Rojos): Diagnósticos científicos sobre el estado de amenaza de las especies. Fundamentan las decisiones legales.
  3. Marcos Regulatorios (Leyes): Herramientas vinculantes que penalizan el tráfico silvestre, protegen ambientes o “hábitats” críticos o regulan recursos nativos.
  4. Dimensión Cultural (Nombres Comunes): El puente directo entre la taxonomía de laboratorio y el saber local/doméstico. Crucial para la participación ciudadana general, la multidisciplina, y el registro de interacciones bióticas comunitarias.
  5. Espacialización de la Conservación (Áreas Protegidas): Delimitación territorial formal de reservas, parques y otras figuras de manejo (WDPA/Protected Planet). Ubica en el mapa dónde la gobernanza ambiental se vuelve jurisdicción concreta.
  6. Planificación del Territorio (Ordenamiento Territorial): Marcos de tenencia, uso y valor de la tierra que condicionan qué convivencia biótica es posible en cada parcela — desde el catastro rural hasta la zonificación urbana.
  7. Horizonte Estratégico (Biodiversidad / NBSAP): Estrategias Nacionales de Biodiversidad y Planes de Acción (Convenio de Diversidad Biológica). El documento político de mayor escala: articula compromisos internacionales con metas nacionales.

Configura los parámetros para extraer el mapa de gobernanza del país seleccionado. Al generar el archivo, estás construyendo un registro transparente bajo principios FAIR/CARE.

Código 
db_ambient_global = FileAttachment("../_data/governance_db.json").json()

// NUEVO: Selector de continente — filtra la lista de países disponible en el paso siguiente.
// "Todos" preserva el comportamiento original (todos los países mezclados).
viewof continente_foco = Inputs.select(
  ["Todos", ...d3.groups(db_ambient_global, d => d.continente).map(g => g[0]).sort()],
  {label: "1. Continente:"}
)

// Lista de países ahora depende reactivamente del continente elegido
paises_disponibles = continente_foco === "Todos"
  ? db_ambient_global
  : db_ambient_global.filter(d => d.continente === continente_foco)

// Selectores reactivos mapeados automáticamente a partir del contenido del JSON
viewof pais_foco = Inputs.select(
  d3.groups(paises_disponibles, d => d.pais).map(g => g[0]),
  {label: "2. País bajo estudio:"}
)

viewof dimensiones = Inputs.checkbox(
  ["Gubernamental", "Libro Rojo", "Leyes", "Nombres Comunes", "Áreas Protegidas", "Ordenamiento Territorial", "Biodiversidad (NBSAP)"],
  {
    label: "3. Dimensiones de análisis:",
    value: ["Gubernamental", "Libro Rojo", "Leyes", "Nombres Comunes", "Áreas Protegidas", "Ordenamiento Territorial", "Biodiversidad (NBSAP)"]
  }
)

Estados de curaduría: manual = enlace curado y verificado puntualmente · plantilla_iso3 = generado automáticamente por código ISO3 (FAOLEX, Protected Planet/WDPA) · plantilla_iso2_provisoria = generado por código ISO2, calidad aproximada (GBIF) · plantilla_slug_provisoria = generado por nombre del país en inglés (Land Portal), verificar que resuelva · manual_pendiente = falta completar la curaduría de esa dimensión para ese país.

Exportación con Trazabilidad Científica

Al presionar el botón de descarga, se inyectarán de forma automatizada los metadatos de autoría de la plataforma directamente en las filas del CSV, asegurando la reproducibilidad del origen de la fuente.

Código 
function crearCSVConMetadatos(data, pais) {
  const metadatos = [
    `# PROYECTO: interdomestic (https://github.com/maordano/interdomestic)`,
    `# ARTEFACTO: Extractor de Gobernanza Ambiental y Vida Silvestre`,
    `# LICENCIA: CC BY 4.0 2026`,
    `# TRAZABILIDAD: Pais evaluado: ${pais} | Fecha de extraccion: ${new Date().toISOString().split('T')[0]}`,
    `# INSTRUCCIONES: Importar en R/Python/Excel omitiendo comentarios que inicien con '#'\n`
  ].join("\n");

  const columnas = "Pais,Continente,Dimension,Recurso,URL,EstadoCuraduria\n";
  const filas = data.map(i => `"${i.pais}","${i.continente ?? ""}","${i.tipo}","${i.recurso}","${i.url}","${i.curaduria ?? "manual"}"`).join("\n");
  
  return metadatos + columnas + filas;
}

// Botón reactivo de descarga directa en local
DOM.download(
  new Blob([crearCSVConMetadatos(datos_filtrados, pais_foco)], {type: "text/csv;charset=utf-8;"}), 
  `interdomestic_${pais_foco.toLowerCase()}_meta.csv`, 
  `📥 Descargar Dataset Estructurado con Trazabilidad (${pais_foco})`
)
