Ficha de protocolo de búsqueda (GSRRF / OSF)

⏳ Primera carga: el entorno estadístico (R) se descarga y arranca completo dentro de tu navegador — esta ficha es liviana (solo shiny), así que debería tardar menos que las otras apps del sitio.

Tip💡 Qué es esta ficha, y qué no es

No es un buscador en vivo (para eso está interacciones.qmd) — es un instrumento de documentación de protocolo. Cubre los 12 ítems de la sección “Search strategy” de la Generalized Systematic Review Registration Form (GSRRF) de OSF, alineada con PRISMA. Los campos vienen pre-completados con un borrador honesto de cómo funciona hoy el motor de búsqueda del proyecto — editalos, corregilos, o marcá “no aplica” donde corresponda; esa es una respuesta válida según las propias instrucciones del formulario.

Las secciones de Screening, Extraction, y Synthesis del GSRRF (33 ítems restantes) quedan para una etapa posterior del proyecto, cuando haya una revisión sistemática real en curso — documentar solo lo que existe hoy, sin inventar procedimientos que todavía no aplican.

A continuación, el formulario de protocolo, ejecutado localmente en su navegador:

# fuente: van den Akker & Peters et al. 2023, Systematic Reviews 12:170
# ==========================================
campos_gsrrf <- list(
  gsrrf21 = list(
    titulo = "GSRRF-21 · Databases",
    prisma = "PRISMA: 6",
    ayuda = "Qué bases de datos se consultan.",
    default = "GBIF (backbone taxonómico y validación de nombres científicos); GloBI - Global Biotic Interactions (registros de interacción documentada con cita de origen); Crossref (metadatos bibliográficos generales, usada como red de contención cuando GloBI no tiene registros para el par de especies consultado)."
  ),
  gsrrf22 = list(
    titulo = "GSRRF-22 · Interfaces",
    prisma = "PRISMA: 6",
    ayuda = "Qué interfaz se usó para acceder a cada base.",
    default = "Acceso directo vía API REST pública de cada base (api.gbif.org, api.globalbioticinteractions.org, api.crossref.org). No se usa una interfaz comercial intermediaria (no Ovid, no EBSCO)."
  ),
  gsrrf23 = list(
    titulo = "GSRRF-23 · Grey literature",
    prisma = "PRISMA: 6",
    ayuda = "Estrategia para localizar literatura gris (preprints, tesis, informes no indexados).",
    default = "No se incluye literatura gris de forma sistemática en esta etapa. GloBI agrega datasets comunitarios (p. ej. iNaturalist) que pueden incluir observaciones no publicadas formalmente en revistas indexadas."
  ),
  gsrrf24 = list(
    titulo = "GSRRF-24 · Inclusion and exclusion criteria",
    prisma = "PRISMA: 5, 13a",
    ayuda = "Criterios que informan la búsqueda y el filtrado de resultados.",
    default = "Se prioriza evidencia que documente explícitamente la interacción entre el par de especies consultado (registro directo en GloBI). Como última red, se acepta literatura que mencione al menos una de las dos especies, marcada explícitamente como de baja confianza."
  ),
  gsrrf25 = list(
    titulo = "GSRRF-25 · Query strings",

    titulo = "GSRRF-26 · Search validation procedure",

    titulo = "GSRRF-28 · Procedures to contact authors",

# ==========================================
# SERVIDOR
# ==========================================
server <- function(input, output, session) {

  output$descargar_protocolo <- downloadHandler(
    filename = function() paste0("interdomestic_protocolo_busqueda_", Sys.Date(), ".md"),
    content = function(file) {
      bloques <- lapply(names(campos_gsrrf), function(id) {
        campo <- campos_gsrrf[[id]]
        valor <- input[[id]]
        paste0("## ", campo$titulo, "\n\n*", campo$prisma, "*\n\n", valor, "\n")
      })
      texto <- paste0(
        "# Protocolo de búsqueda — interdomestic\n\n",
        "Ficha generada: ", format(Sys.time(), "%Y-%m-%d %H:%M %Z"), "\n\n",
        "Alineada con la sección \"Search strategy\" de la Generalized Systematic Review Registration Form (OSF/PRISMA).\n\n",
        paste(bloques, collapse = "\n")
      )
      writeLines(texto, file)
    }
  )
}

shinyApp(ui, server)
