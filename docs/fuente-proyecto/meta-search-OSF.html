Tip💡 Qué es esta ficha, y qué no es

No es un buscador en vivo (para eso está Interacciones Bióticas) — es un instrumento de documentación de protocolo. Cubre los 12 ítems de la sección “Search strategy” de la Generalized Systematic Review Registration Form (GSRRF) de OSF, alineada con PRISMA. Los campos vienen pre-completados con un borrador honesto de cómo funciona hoy el motor de búsqueda del proyecto — editalos, corregilos, o marcá “no aplica” donde corresponda; esa es una respuesta válida según las propias instrucciones del formulario.

Las secciones de Screening, Extraction, y Synthesis del GSRRF (33 ítems restantes) quedan para una etapa posterior del proyecto, cuando haya una revisión sistemática real en curso — documentar solo lo que existe hoy, sin inventar procedimientos que todavía no aplican.

NotaSobre la ubicación de esta ficha

Esta ficha vive en Fuente de proyecto porque, a futuro, reflejará la búsqueda tal como quede definida en el propio registro OSF del proyecto macro — compartiendo desde ya la misma membrana digital que el resto de los dispositivos de interdomestic (ver Dispositivos de Interacción). Todavía falta la conceptualización y la complementación con los otros componentes del proyecto; por ahora, esta es una ficha editable y descargable, no una integración automática con OSF.

Código 
function generarProtocoloMD(valores) {
  const fecha = new Date().toLocaleString('es-AR')
  const bloques = campos_gsrrf.map(c =>
    `## ${c.titulo}\n\n*${c.prisma}*\n\n${valores[c.id]}\n`
  ).join("\n")
  return `# Protocolo de búsqueda — interdomestic

Ficha generada: ${fecha}

Alineada con la sección "Search strategy" de la Generalized Systematic Review Registration Form (OSF/PRISMA).

${bloques}`
}

//| output: false

function descargarProtocolo() {
  try {
    const md = generarProtocoloMD(valores_gsrrf)
    const blob = new Blob([md], {type: "text/markdown;charset=utf-8;"})
    const url = URL.createObjectURL(blob)
    const link = document.createElement("a")
    link.href = url
    link.download = `interdomestic_protocolo_busqueda_${new Date().toISOString().split('T')[0]}.md`
    link.style.display = "none"
    document.body.appendChild(link)
    link.click()
    document.body.removeChild(link)
    // Revocar el blob con demora: revocarlo de inmediato es una causa
    // conocida de descargas cortadas a mitad de camino en Chromium,
    // porque .click() dispara la descarga de forma asíncrona.
    setTimeout(() => URL.revokeObjectURL(url), 1000)
    console.log("descargarProtocolo: OK")
  } catch (error) {
    console.error("descargarProtocolo: FALLÓ ->", error)
    alert("No se pudo generar la descarga: " + error.message)
  }
}

//| echo: false
viewof disparador_descarga = {

    value: null,
    reduce: () => descargarProtocolo()
  })
  form.querySelector("button").classList.add("bio-button", "bio-button-generate")
  return form
}
